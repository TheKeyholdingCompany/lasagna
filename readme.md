<h1 align="center">Lasagna</h1>

A CLI tool to build your lambda layer zip for you.

# Usage
```
Usage:
  lasagna [options]
  lasagna --version
  lasagna --help

Options:
  -h --help           Show this Help.
  -o --output=<path>  Path to output file [default: ./layer.zip].

Examples:
  lasagna --output=./my-layer.zip
```
